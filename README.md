## About Project
Hexlet "Brain-Games" project.

## Install project
- `make install` for install dependencies (required php and other default libraries).

## Composer
- `make validate` for composer.json validation.
- `make dump` for composer dump autoload.

## Linters and tests
- `make lint` for linter.
